## 📖 Overview
This section contains my additional Java practice during the summer after my first year (2024–2025).

It focuses on **reinforcing core concepts**, improving problem-solving skills, and exploring topics more deeply due to limited time during the regular semester.

---

## 📚 Topics Covered

### ⚙️ Methods (Reinforcement)
- Continued practice of method creation and usage  
- Applying the four types of methods in different scenarios:
  - Without parameters, without return value  
  - With parameters, without return value  
  - Without parameters, with return value  
  - With parameters, with return value  

---

### 📦 Arrays (Deeper Learning)
- Multi-dimensional arrays (2D arrays)  
- Storing and organizing data in table-like structures  
- Traversing arrays using nested loops  

---

### 🧩 Problem-Solving Focus
- Understanding problems before coding  
- Analyzing inputs, processes, and expected outputs  
- Applying logic step-by-step to create meaningful solutions  

> A key lesson learned:  
> Being a programmer is not just about writing code, but about **solving real problems**.  
> Without problem-solving skills, programs can become meaningless or redundant.
